2.3 Embedded Tags

HTML and XHTML are embedded languages which are a collection of tags/

Tags are directions that are within a document that instructs the browsers on how to read and/or display the content of the document

The first word in a tag is its formal name, which is the description of the function.

Additional keywords within the tag are special attributes

Attributes are followed with an equals sign (=), which further define or modify the tag's actions

Keywords
